# Top, Ps, Jobs, Kill

1.top
-------------
top 명령어는 현재 OS의 상태를 나타내는 명령어이다.
화면 상단에는 시스템 요약 정보가, 하단에는 프로세스 목록이 표시된다.

OS의 상태 중에서, CPU와 메모리의 상태를 실시간으로 확인 가능하며, 실행중인 프로세스의 목록을 표시할 수 있다.

|항목|설명|
|---|---|
|PID|프로세스 ID|
|USER|사용자|
|%CPU|CPU 사용률|
|%MEM|메모리 사용률|
|COMMAND|실행 명령어|


2.ps
-------------
ps 명령어는 현재 실행중인 프로세스를 출력하는 명령어이다.

|옵션|설명|
|---|---|
| ps |현재 사용자가 실행한 터미널의 프로세스만 출력 |
| ps -ef | 시스템에서 실행 중인 모든 프로세스를 상세하게 출력 |
|ps aux| 모든 프로세스를 사용자 및 자원 사용량과 함께 출력|

특정 시점의 프로세스 상태를 출력할 수 있으며, 위의 옵션처럼 다양한 옵션으로 상세 정보를 확인할 수 있다.


3.jobs
----------
jobs 명령어는 백그라운드에서 실행 중인 작업 목록을 보여준다.
현재 쉘에서 실행된 작업만 표시되며, 작업 상태 또한 확인할 수 있다.

예를 들어, 출력이 "[1] + Running  sleep 100 &" 이라면,
|출력|설명|
|---|---|
|[1]|작업 번호|
|Running|상태|
|sleep 100|실행 명령|

을 나타낸다.


4.kill
-----------
kill 명령어는 프로세스를 종료하거나 시그널을 보내는 명령어이다.

'kill [signal] [pid]'

형식을 사용하여 시그널을 보낸다.

|시그널 이름|옵션|설명|
|---|---|---|
|SIGTERM|-15|정상 종료 (정상적으로 저장 후 프로그램 종료)|
|SIGKILL|-9|강제 종료 (즉시 프로그램 종료)|
|SIGHUP|-1|재시작/재적용 (프로그램 종료 없이 설정 파일을 다시 불러옴)|
